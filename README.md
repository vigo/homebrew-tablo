# homebrew-tablo

Homebrew tap for [tablo](https://github.com/vigo/tablo) — a CLI data pipeline tool written in Go.

## Installation

```bash
brew tap vigo/tablo
brew install tablo
```

## Update

```bash
brew update
brew upgrade tablo
```
